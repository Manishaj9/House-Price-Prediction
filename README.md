# Kaggle Housing Prices Model

This repository contains my first machine learning project, completed through the Kaggle **Intro to Machine Learning** course. 

## 📌 Project Overview
The goal of this project is to predict housing prices using key features like lot size, room count, and the year the home was built.

## 🛠️ Concepts Applied
* Data exploration using **pandas**
* Feature engineering and target selection
* Model training using a **Decision Tree Regressor** via scikit-learn
* Model evaluation using **Mean Absolute Error (MAE)**

## 🚀 How to Run
1. Download the notebook file.
2. Upload it to Kaggle or run it locally using Jupyter Notebook.
